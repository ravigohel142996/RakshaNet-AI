# MVP Scope Lock

## In Scope
1. SOS wearable button press from registered user band.
2. Nearest smart pole alert over LoRa.
3. Pole siren + flashing light trigger.
4. GSM escalation to 3 trusted contacts.
5. Basic nearest-helper routing recommendation.

## Out of Scope (Phase 2+)
- Full city-wide command center integration.
- Predictive threat analytics.
- Real-time video analytics model on edge.

## Success Criteria
- SOS acknowledgment under 5 seconds in local network.
- Alert fan-out to 3 contacts under 30 seconds via GSM.
- Incident captured with traceable timeline and location.
